# InsightForge – AI-Powered Business Intelligence Assistant

## 1) Problem & Goal
SMBs often collect data (sales, customers, operations) but can’t afford enterprise BI. InsightForge uses LLM + RAG to turn business data into natural language insights and visuals: trends, anomalies, and recommendations.

## 2) Solution Architecture

### A. Data Pipeline
- **Ingest**: CSV/Excel exports, system logs, and reference docs (policies, goals).
- **Clean/normalize**: Schema checks, missing-value strategy, consistent time zones, standardized dimensions (e.g., region/category).
- **Feature store**: Create KPIs (YoY/DoD, churn proxies, revenue per unit/customer, conversion rates).
- **Views**: Curated summary tables for fast retrieval and visualization.

### B. RAG Pipeline
- **Chunking**: Structured insights (KPI snapshots) + unstructured docs (strategy docs).
- **Embeddings**: Turn chunks into vectors.
- **Vector store**: FAISS/Chroma for retrieval.
- **Retriever + LLM**: User question → retrieve relevant chunks → LLM generates answer with references and caveats.

### C. Visualization
Streamlit/Gradio dashboard:
- Top KPIs card
- Trend lines for revenue/orders
- Pareto chart for top contributors
- Segmentation breakdown (region, category)
- “Insight cards” generated by LLM

## 3) Analysis & Insights (Template)
- **Trend**: Revenue/volume vs prior period → identify peaks/slowdowns.
- **Top drivers**: Which categories/products/regions contribute most (Pareto 80/20).
- **Customer behavior**: Retention proxy, repeat rate, cohort curve.
- **Operational alerts**: Spikes in refunds/cancellations, supply gaps.
- **Recommendations**:
  - Double down on top 2 growth categories and underperforming regions with clear campaigns.
  - Fix the top “loss driver” metric (e.g., high cancellation segments).
  - Raise conversion rate via upsell bundles and checkout optimization.

## 4) Evaluation
- **Ground truth QA**: Compare LLM answers against curated SQL summaries.
- **Answer quality metrics**: Grounding (uses cited chunks), correctness, coverage, clarity.
- **Performance**: Latency, retrieval hit rate, cache effectiveness.

## 5) Security & Governance
- PII tokenization/column masking.
- Tenant isolation for multi-org deployments.
- Activity logging and prompt auditing.
