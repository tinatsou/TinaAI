# InsightForge Screenshots

This folder contains placeholder images for the InsightForge dashboard, example query, and insight cards.
Since the live Streamlit app relies on real data or an active API endpoint to generate the visualizations, these placeholders have been included so the folder structure is complete.

## How to replace with real screenshots:

1. Run the Streamlit app locally: `streamlit run src/query_app.py`
2. Take a screenshot of the top half of the dashboard (KPI cards, trend line, top contributors). Save it as `dashboard_insightforge.png`.
3. Enter the query "Which region drove growth last quarter?" into the app, generate the answer, and take a screenshot. Save it as `query_growth_region.png`.
4. Take a screenshot of the "Insight Cards" section showing the recommended actions. Save it as `insight_cards.png`.
5. Replace the placeholder files in this folder with your actual screenshots.
